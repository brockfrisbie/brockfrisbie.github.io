I've been looking for an easy way to get a blog up and running to keep track of some of the projects I've been working on.  I 
stumbled onto Jekyll while reading the [Learn Enough CSS & Layout Tutorial](https://www.learnenough.com/css-and-layout-tutorial
-tutorial).  After reading a bit on Jekyll there I decided to give it a try.  I had to pick my way through the tutorail to find
the pieces I needed to get a simple blog up and running.  

I decided to build my Jekyll project in a [Cloud 9](https://c9.io) environment. I've used Cloud 9 a lot recently for learning Ruby
on Rails and for working through some problem sets for [Harvard's CS50 course](https://cs50.harvard.edu), and I really like it.  The
Jekyll project is running in a blank workspace, which comes with Ruby 2.4.0 by default.  All I had to do was install Jekyll, and I 
was ready to get started with a new project.
